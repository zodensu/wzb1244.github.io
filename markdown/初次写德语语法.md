# 说明
   时间连词**als**和**wenn表示“当……时候”，als**通常用于描写过去某时间一次性发生的
   事，所以时态要求用过去时。其他情况一律用连词**wenn**，时态要根据具体情况而定。
   为了强调多次发生，可以在**wenn**前加上**jedes Mal**或**immmer**。见下表：

    | |过去|现在/ 将来|
    |:---:|:---:|:---:|
    |一次性|**als**|wenn（将来）|
    |多次性|wenn|wenn（现在）|
